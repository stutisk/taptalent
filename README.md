## Weather Dashboard

A React weather app that shows weather for multiple cities and allows users to add/remove favorite cities.

### Features
- View weather for all cities
- Add / remove favorite cities
- Favorites stored using Redux + localStorage
- City search with smooth scroll to top
- Toast notifications on favorite actions
- Unit toggle (°C / °F)

### Tech Stack
- React
- Redux Toolkit
- Tailwind CSS
- React Toastify
- OpenWeather API
